# developer-card-v2

A frontend practice project built while learning HTML, CSS, GitHub, and StackBlitz.

This project is the second iteration of my original Developer Card project and focuses on improving layout structure, visual styling, spacing, and overall UI presentation.

## Technologies Used
- HTML
- CSS
- GitHub
- StackBlitz

## What I Practiced
- Building a centered card layout
- Styling profile images
- Creating gradient backgrounds
- Using hover effects and transitions
- Improving spacing and alignment
- Structuring sections more cleanly
- Using GitHub for version control

## Project Goal
The goal of this project was to become more comfortable building and improving frontend layouts while learning real developer workflows and project iteration.
